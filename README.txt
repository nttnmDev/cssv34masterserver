Данный репозиторий содержит файл masterservers.vdf,
который задаëт главный сервер для поиска всех серверов.
Публичный адрес мастерсервера: 93.115.101.178:10232
На данный момент мастерсервер содержит около 310 активных серверов,
собирая их с нескольких сайтов: tsarvar.com, myarena.ru, monitoring-css.ru.

Установка:
1. Скачать файл с репозитория
2. Закинуть в <папка с игрой>/platform/config
(Рекомендуется установить файлу атрибут "Только чтение",
чтобы игра не удаляла содержимое при неудачной попытке подключиться к мастерсерверу)
3. Перезапустить игру
